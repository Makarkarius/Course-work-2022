Information

Aho–Corasick algorithm is a string-searching algorithm that locates all entries of elements of a finite set of strings (the "dictionary") within an input text.

Algorithm constructs a finite-state machine that resembles a trie with additional suffix links between the various internal nodes to other branches of the trie that share a common prefix. These links allow fast transitions between failed string matches.

Complexity

If storing transitions as an array:
Memory: O(n * δ)
Search: O(nδ + H + k)

If storing transitions as R-B Tree:
Memory: O(n)
Search: O((H+n)log δ + k)

H - text length, n - summary length of words in dictionary, δ - alphabet size, k - summary length of word matches.
